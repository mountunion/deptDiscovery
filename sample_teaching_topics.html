Topics for Sample Teaching Lesson:

Please prepare a sample lesson of 40 to 50 minutes on one of these topics that is typically part of a CS2 or Data Structures undergraduate course:

 		   
    ☆   Two-Dimensional (or multi-dimensional) Arrays

    ☆   Recursive functions (using an example other than the factorial function)

    ☆   A Recursive Sorting algorithm (MergeSort, QuickSort, HeapSort, etc.)

    ☆   Linked Lists

    ☆   Stacks and/or Queues

    ☆   Priority Queues

    ☆   Hash Functions and Hash Tables

    


The lesson should include a discussion of the topic and code for implementing it.  A live coding demo would be appropriate, or results of a program run on sample data could be shown.
