CSW Course Descriptions:

Course Number and Title: Course Description:
CSW 123   Web Development Fundamentals In this course, students will learn the fundamentals of Hypertext Markup Language (HTML) and Cascading Style Sheets (CSS) to develop websites.  Throughout this course, students will apply visual design principles when creating professional websites that are accessible and follow universal design principles.
CSW 223   Programming for Interactive Media Implementation of industry-standard programming concepts using modern web programming languages and frameworks.  Concepts include tools and techniques for development of web content, with emphasis on coding and interactivity.  4 Semester Hours.

Prerequisite: CSW 123.
CSW 423   Interaction Design This course is concerned with the design, evaluation and implementation of interactive computing systems for human use and with the study of major phenomena surrounding them. Basic design theory from both art and computer science will be studied. 4 Sem. Hrs.

Prerequisite: CSW 123.
