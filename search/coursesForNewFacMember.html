Department of Computer Science

CSC Courses that Must be taught by New Faculty Member in 2021-'22:

(4 courses: 16 credit hours)

Course Number and Title: Course Description:
CSC 310   Database Theory and Implementation

        4 credit hours 		A theoretical introduction to database models, database design, normalization and data administration. Specific applications are studied and developed using fourth generation languages. 4 Sem. Hrs.

Prerequisite: CSC 220.
CSC 320   Algorithms and Data Structures

        [Spring 2022]
  		Topics covered include linked lists, stacks, queues, trees, recursion, searching, sorting, hashing, and analysis and measurement of algorithms. 4 Sem. Hrs.

Prerequisites: CSC 220, MTH 125 and MTH 141.
CSC 360   Computer Networks

        [Fall 2021] 		This course provides an introduction to the technologies used in computer networks. Topics include network architecture, network protocols, and software and hardware implementations of the protocols. 4 Sem. Hrs.

Prerequisite: CSC 220 or CSC 270.
CSC 370   Operating Systems

        [Fall 2021] 		The study of software designed to control the operation of the components of a computer system. A survey of typical operating systems is included along with investigation of concurrent processes, deadlock, memory management, file systems, processor scheduling/utilization, and distributed and/or parallel systems. Programming skills will be utilized and expanded. 4 Sem. Hrs.

Prerequisites: CSC 220 and CSC 270.
CSC 420   Principles of Programming Languages

        [Spring 2022] 		A study of the principles of the design and implementation of computer programming languages. Topics include syntax, lexical analysis, Backus-Naur Form, parsing, compilers, interpreters, binding, and the run-time environment. Languages of various types are examined. 4 Sem. Hrs.

Prerequisites: CSC 220 and MTH 125.


CSC Courses that COULD be taught by New Faculty Member in 2021-'22:

to fill out the Faculty Member's schedule

would need 8 credit hours from this list (4 in Fall 2021, 4 in Spring 2022):

Course Number and Title: Course Description:
CSC 105   Introduction to Linux

        2 credit hours 		This course provides an introduction to the Linux operating system, from a user's perspective. Topics include installation, software installation and updates, security issues, network configuration, file systems, graphical and command line interfaces, and shell programming to automate repetitive tasks. 2 Sem. Hrs.
CSC 140   Data Science Fundamentals

        4 credit hours 		This course provides an introduction to data science, including importing and exporting data, data cleaning and preparation, visualization and statistical analysis, and basic programming skills in programming languages commonly used in the data science field. It will also explore strengths and limitations of modern data analysis, and some of the ethical issues that emerge from the use of data science. Students will gain a general understanding of data science terms, approaches to problems, and strategies for effectively using data science. 4 credit hours.
CSC 226   Intermediate Programming in Scala

        2 credit hours 		A course in programming techniques using the Scala programming language, intended for students who have previously taken several programming courses. Prerequisite: CSC 220. 2 Semester Hours.
CSC 228   Intermediate Programming in Kotlin

        2 credit hours 		A course in programming techniques using the Kotlin programming language, intended for students who have previously taken several programming courses. Prerequisite: CSC 220. 2 Semester Hours.
CSC 290   Professional Programming Practices

        2 credit hours 		This course provides practical instruction in various software development techniques and approaches to problem solving that are commonly employed by software and tech firms. Students will learn about software testing strategies, configuration management systems, build systems, and product deployment concepts. Course content will be supplemented with hands-on projects that give students experience with various software development tools and techniques. Prerequisite: CSC 220. 2 Semester Hours.
CSC 430   Programming Parallel Computing Systems

        4 credit hours 		This course provides the student with an in-depth introduction to programming parallel computer systems. Major topics include the taxonomy of parallel systems and parallel algorithms. Programming assignments on GPU systems will provide hands-on experience. 4 Sem. Hrs.

Prerequisites: MTH 125 and CSC 220.
CSC 470   Computer Architecture and Design

        4 credit hours 		An examination of design principles and techniques used in contemporary microprocessors and computers to achieve high performance. Topics include pipelining, caching, parallelism, code optimization, and case studies of real-world systems. 4 Sem. Hrs.

Prerequisites: CSC 220 and CSC 270.
CSC 480   Artificial Intelligence

        4 credit hours 		An overview of the field of Artificial Intelligence, including: knowledge representation, logical systems, forward and backward reasoning, searching, learning, planning, natural language processing, case- and rule-based systems, and genetic algorithms. 4 Sem. Hrs.

Prerequisite: CSC 220.
CSC 399   Special Topics

        4 credit hours 		A course designed to permit the offering of special subjects appropriate to the program of the department.  Such offerings will fill special needs of specific students, take advantage of the expertise of a visiting or new professor, or serve as an initial experimental offering of a contemplated regular course.  Upper divisional offerings will be listed as 399.  4 Sem. Hrs.
